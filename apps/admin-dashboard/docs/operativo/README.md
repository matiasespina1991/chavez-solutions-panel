# Guía Operativa (No técnica)

Esta documentación explica **cómo funciona el proceso de trabajo** en lenguaje operativo, sin entrar en implementación técnica.

## ¿Para quién es?

- Equipo comercial
- Equipo operativo
- Coordinación / supervisión
- Personas nuevas que se incorporan al flujo

## Qué vas a encontrar

- [01. Flujo general de trabajo](./01-flujo-general.md)
- [02. Qué hacer en cada pantalla](./02-por-pantalla.md)
- [03. Reglas y casos comunes](./03-reglas-y-casos.md)

## Resultado esperado

Al terminar esta guía, deberías poder:

- Crear una solicitud correctamente.
- Entender cuándo emitir una OT.
- Saber cuándo pausar, reanudar o finalizar.
- Evitar acciones que rompan el flujo operativo.
