# 02. Qué hacer en cada pantalla

## 1) Configurador de proformas y OT

Usar esta pantalla cuando querés **crear** una solicitud nueva o **editar** una existente permitida.

### Acciones principales

- Cargar datos de cliente, muestras, análisis y notas.
- Guardar como borrador (para continuar luego).
- Ejecutar proforma.
- Ejecutar proforma + OT.

### Cuándo usar cada acción

- **Guardar borrador**: cuando la información está incompleta o pendiente de validación.
- **Ejecutar proforma**: cuando querés emitir propuesta sin iniciar trabajo operativo.
- **Ejecutar proforma + OT**: cuando ya se aprueba iniciar la operación.

---

## 2) Lista de solicitudes

Es la bandeja para **gestionar solicitudes** después de creadas.

### Para qué se usa

- Buscar y revisar solicitudes.
- Ver detalles completos.
- Editar solicitudes cuando el estado lo permite.
- Emitir OT desde una solicitud.
- Pausar/Reanudar OT.
- Eliminar solicitudes.

### Recomendación operativa

Revisar esta lista todos los días para detectar:

- borradores pendientes,
- proformas listas para avanzar,
- OT pausadas que requieren seguimiento.

---

## 3) Lista de órdenes de trabajo

Es la bandeja para **seguimiento y cierre** de OT emitidas.

### Para qué se usa

- Monitorear estado de OT en curso.
- Abrir detalle de cada OT.
- Finalizar OT cuando el trabajo terminó.

### Punto importante

Una OT finalizada debe quedar cerrada y no volver al circuito operativo normal, salvo políticas de excepción definidas por el negocio.
